# Music API

A RESTful backend application for music and playlist management built with Node.js, Express.js, and SQLite.

## Features

- User Registration & Login
- JWT Authentication
- Password Hashing with bcrypt
- Search Music and Artists
- Playlist Management
- Add & Remove Songs from Playlists
- Logging with Winston
- SQLite Database
- RESTful API

## Tech Stack

- JavaScript
- Node.js
- Express.js
- SQLite
- JWT
- bcrypt
- dotenv
- Docker
